Как открыть:
1. Дважды кликните по rimini-guide.html.
2. Файл откроется в браузере.
3. Ссылки на карты, билеты и официальные сайты открываются из документа.

Как поделиться (файлом):
1. Заархивируйте папку site\ (в ней только нужное для гайда) или всю rimini-travel-guide.
2. Получатель распаковывает и открывает index.html.
3. Если отправить только .html без папки images — фотографии не отобразятся.
4. Для PDF: откройте в браузере → Ctrl+P → «Сохранить как PDF».

СПОСОБ 1 — Netlify Drop (самый быстрый, ~2 минуты, бесплатно):
1. Откройте https://app.netlify.com/drop в браузере.
2. Перетащите папку site в окно (именно site, не весь проект).
3. Netlify выдаст ссылку вида https://случайное-имя.netlify.app
4. Гайд откроется сразу, все фото будут работать.
5. Site configuration → Domain management → можно переименовать (rimini-guide-2026.netlify.app).

СПОСОБ 2 — GitHub Pages (постоянная ссылка, если есть GitHub):
1. Создайте репозиторий на github.com (например rimini-travel-guide).
2. Загрузите файлы: index.html, rimini-guide.html, папку images/ (служебные .ps1 не нужны).
3. Settings → Pages → Source: Deploy from branch → main → / (root) → Save.
4. Через 1–2 мин сайт будет на https://ВАШ-ЛОГИН.github.io/rimini-travel-guide/

СПОСОБ 3 — Google Drive / Dropbox (не идеально):
Обычный просмотр HTML в облаке часто ломает пути к картинкам.
Лучше использовать Способ 1 или 2. ZIP для пересылки в мессенджере — ок.
